FAQ

Common questions about the live workflow.

Can citizens still submit complaints exactly the same way? +

Yes. The visual design has changed, but the core complaint flow, AI analysis, PDF generation, BBMP email, map preview, and admin permissions are preserved.

Does the admin dashboard still control alerts, status updates, and reset? +

Yes. Admin-only actions remain connected to the same permission-based backend routes. This redesign changes presentation, not authorization logic.

What changed for voice input? +

Voice mode now supports audio-file upload and backend transcription, which is more reliable across devices than browser speech recognition.
